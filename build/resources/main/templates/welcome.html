Welcome to The Hangman Game!

Rules of the game:

  • Click on the Start New Game button to start a new game
  • A new word is selected when you do
  • You can guess a correct letter multiple times, but it will be added to the number of guesses
  • Each time an incorrect letter is guessed, it will be counted toward the number of incorrect guesses, regardless of whether this letter has been guessed before.
  • Empty guesses are not counted toward any guesses.
  • You are allowed up to 7 incorrect guesses.
  • Only 1 letter guesses will be valid.
  • Enjoy my computer-hand drawn hangman images.
